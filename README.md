# Summit Resolutions

Track daily rituals, monthly milestones, and anytime goals with a calm mountain theme.

## Features

- Daily, monthly, and anytime resolution lists
- Target counts, notes, and optional due dates
- One-click check-ins with progress meters
- Local browser storage (no backend required)

## Getting Started

Run the development server:

```bash
npm run dev
```

Then visit http://localhost:3000.

## Scripts

- `npm run dev` - Start the local dev server
- `npm run build` - Production build
- `npm run start` - Serve the production build
- `npm run lint` - Lint the codebase
